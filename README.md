# Smart Energy Automation System

## Overview

This project demonstrates a distributed smart energy management system built in Java.

The system uses:

- Java
- Maven
- gRPC
- Protocol Buffers
- jmDNS Service Discovery
- Git & GitHub

The goal is to simulate multiple smart devices communicating over a network to improve energy efficiency in buildings.
